# FocusMate
# En

FocusMate — a Telegram bot for concentration and Pomodoro tracking.

## Functions
- /remind <duration> <text> - set a reminder 
- /focus <duration> — start a focus session, notifications every 2 minutes
- /stopfocus — stop the active focus session
- /pomodoro — start a Pomodoro cycle: 25 minutes work → 5 minutes break → repeat
- /pomodorocancel — stop the Pomodoro cycle
